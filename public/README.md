# Adding Screenshots - Behavior Simulation for Parents

To display your laptop mockup screenshots in the Behavior Simulation showcase section:

1. Save your screenshot images as:
   - `screenshot1.png` - Scenario Simulation Engine interface
   - `screenshot2.png` - Real-Time Guidance Dashboard

2. Place these files in the `/public` folder of your project

3. The images will automatically appear in the laptop frame mockup showcasing the Behavior Simulation for Parents product

## Image Specifications

- **Format**: PNG (recommended for transparency and quality)
- **Resolution**: 1920x1080 or higher recommended
- **Aspect Ratio**: 16:9 or 16:10 (laptop screen proportions)
- **Content**: Screenshots from your Behavior Simulation dashboard showing environment simulations, stress predictions, and parent guidance

## Fallback

If the images are not found, the component will display a gradient background as a placeholder.
